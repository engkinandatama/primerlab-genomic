## Prerequisites

Before starting, ensure you have:

1. **PrimerLab installed:**

   ```bash
   pip install primerlab-genomic
   ```

2. **Dependencies verified:**

   ```bash
   primerlab health
   ```

3. **A target sequence** in FASTA format
