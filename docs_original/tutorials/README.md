## Before You Start

1. Install PrimerLab:

   ```bash
   pip install git+https://github.com/engkinandatama/primerlab-genomic.git@v0.3.5
   ```

2. Verify installation:

   ```bash
   primerlab --version
   primerlab health
   ```
